# 🍽️ HungryHub Dashboard

HungryHub is a modern food ordering dashboard that provides **smart food recommendations** based on:
- 🌦 Weather conditions
- 😊 User mood
- 🔥 Calorie tracking

---

## 🚀 Features

### 🎯 Smart Recommendations
- Suggests food based on **weather + mood**
- AI-like filtering system

### 🍔 Food Catalog
- Multiple categories: Pizza, Drinks, Indian, Healthy, etc.
- Each item includes:
  - Price 💰
  - Calories 🔥
  - Tags (Low-calorie, High-protein)

### 🧠 Mood-Based Filtering
- Happy 😊
- Stressed 😮‍💨
- Lazy 🛋
- Energetic ⚡

### 🌦 Weather Integration
- Adjusts recommendations based on:
  - Hot ☀️
  - Cold ❄️
  - Rainy 🌧
  - Moderate 🌤

### 🛒 Cart & Wishlist
- Add/remove items
- Persistent wishlist using localStorage

### 🎨 Theme System
- Multiple themes:
  - Light
  - Dark
  - Forest
  - Ocean
  - Rose
  - Midnight
